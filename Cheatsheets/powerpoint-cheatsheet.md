# Microsoft PowerPoint Cheatsheet

## Navigation & View Shortcuts

```text
Ctrl+M                    New slide
Ctrl+D                       Duplicate selected slide/object
Page Up / Page Down             Previous / next slide (in editing view)
Ctrl+Home / Ctrl+End                Jump to first / last slide
Alt+Ctrl+Tab or View tab               Switch between Normal, Slide Sorter, Notes Page, Outline, Reading View
Shift+F5                                  Start slideshow from current slide
F5                                            Start slideshow from the beginning
Esc                                              Exit slideshow
Ctrl+A (in Slide Sorter)                            Select all slides
```

## During a Slideshow (Presenter Controls)

```text
N / Enter / Space / →       Next slide/animation step
P / Backspace / ←               Previous slide/animation step
Number + Enter                     Jump to a specific slide number
B                                      Black out the screen (press again to resume)
W                                         White out the screen
Ctrl+P                                       Turn cursor into a pen (annotate live)
Ctrl+E                                          Turn cursor into an eraser
Ctrl+A                                             Turn annotation tool back into a normal pointer
E                                                     Erase all ink annotations on the current slide
Ctrl+H                                                  Hide pointer/annotation and navigation UI immediately
S                                                          Stop/restart an automatic slideshow
G                                                             Show slide grid / zoom to Slide Sorter mid-show (varies by version)
Tab                                                              Show all clickable objects/hyperlinks on the slide
Alt+F5                                                              Open Presenter View
```

## Slide Design & Layout

```text
Home > New Slide > choose a layout (Title, Title+Content, Two Content, Comparison, Blank, etc.)
Design tab > Themes — apply a consistent visual theme across the deck
Design tab > Variants — color/font variations within a theme
Design tab > Slide Size — Standard (4:3) vs Widescreen (16:9), custom dimensions
Design tab > Format Background — solid/gradient/picture/pattern background, per-slide or applied to all

View > Slide Master — edit the master layout that propagates to all slides using that layout
  Insert Slide Master / Insert Layout to build custom, reusable layouts
  Placeholders (Title, Content, etc.) defined here control what appears in Normal view

View > Guides / Gridlines (Shift+F9 / Alt+F9)     alignment aids
Arrange > Align (left/center/right/top/middle/bottom, distribute horizontally/vertically)
Arrange > Group / Ungroup (Ctrl+G / Ctrl+Shift+G)
Arrange > Bring Forward / Send Backward (Ctrl+Shift+] / Ctrl+Shift+[)
```

## Inserting Content

```text
Insert > Text Box
Insert > Picture (from file, stock images, online pictures, screenshot)
Insert > Shapes / Icons / 3D Models / SmartArt
Insert > Chart — embeds an editable Excel-backed chart
Insert > Table
Insert > Video/Audio (from file or online)
Insert > Screen Recording
Insert > Symbol / Equation (Alt+=)
Insert > Header & Footer — slide number, date, footer text across all/selected slides
Insert > Hyperlink (Ctrl+K) — link to a URL, another slide, email, or file
Insert > Action — clickable object that jumps to a slide, runs a program, or plays a sound on click/hover
```

## Text Formatting

```text
Ctrl+B / Ctrl+I / Ctrl+U        Bold / Italic / Underline
Ctrl+E / Ctrl+L / Ctrl+R                Center / Left / Right align text
Ctrl+] / Ctrl+[                            Increase / decrease font size
Ctrl+Shift+C / Ctrl+Shift+V                   Copy / paste formatting (Format Painter)
Tab / Shift+Tab (in outline/bullet text)         Demote / promote bullet level
Alt+Shift+→ / ← (in outline)                        Demote / promote a whole section
Ctrl+Enter                                             Move to next placeholder / add new slide if on last
```

## Animations

```text
Animations tab > Add Animation
  Entrance (Appear, Fade, Fly In, Wipe, Zoom, etc.)
  Emphasis (Pulse, Grow/Shrink, Spin, Color change)
  Exit (mirrors entrance effects, object leaves the slide)
  Motion Paths (custom or preset movement across the slide)

Animation Pane (Animations > Animation Pane) — see, reorder, and time all animations on a slide
Effect Options — customize direction/shape/sequence per animation
Timing group:
  Start: On Click / With Previous / After Previous
  Duration, Delay
  Trigger — start an animation when a specific object is clicked

Animation Painter — copy animations from one object to another (like Format Painter for animations)
```

## Slide Transitions

```text
Transitions tab > choose a transition (Fade, Push, Wipe, Morph, etc.)
Effect Options — direction/style variant of the chosen transition
Timing group:
  Sound, Duration
  Advance Slide: On Mouse Click / After [time] (for auto-advancing/self-running presentations)
Apply To All — apply the current transition settings to every slide

Morph transition — smoothly animates matching objects between two slides (duplicate a slide, move/resize objects, apply Morph)
```

## Slide Master, Layouts & Themes (Deeper)

```text
View > Slide Master
  Top-left largest thumbnail = Slide Master (governs ALL layouts below it)
  Each layout beneath it = a specific slide type (Title Slide, Title and Content, etc.)
  Edit fonts/colors/placeholders on the Slide Master to change them everywhere at once
  Insert Placeholder — add custom placeholder types to a layout (text, picture, chart, etc.)

Design > Format Background > Apply to All      set a background across the whole deck at once
Slide Master > Colors / Fonts / Effects dropdowns — swap theme color/font sets without leaving Master view
```

## Speaker Notes & Presenter View

```text
View > Notes Page / click "Notes" at bottom of Normal view — add speaker notes per slide
Slide Show > Presenter View — shows current slide, next slide, notes, and a timer on your screen
  while the audience only sees the current slide on the projector/shared screen
Slide Show > Rehearse Timings — practice and record per-slide timing for a self-running deck
Slide Show > Record Slide Show — record narration, ink, and webcam alongside slide timings
```

## Working with Charts & SmartArt

```text
Insert > Chart > choose type > edit data in the linked mini-Excel sheet
Chart Design tab > Change Chart Type, Chart Styles, Switch Row/Column, Edit Data
Chart Elements (+ button next to selected chart) — toggle axis titles, data labels, gridlines, legend

Insert > SmartArt > choose a diagram category (List, Process, Cycle, Hierarchy, Relationship, Matrix, Pyramid)
SmartArt Design tab > Add Shape, Change Colors, SmartArt Styles
Convert existing bulleted text to SmartArt: select text box > Home > Convert to SmartArt
```

## Collaboration & Review

```text
Review > Compare — compare two versions of a presentation, accept/reject changes

Slide Show > Set Up Slide Show > Browsed by an individual (window) — useful for testing without full-screen
```

## Exporting & Output Formats

```text
File > Export > Create PDF/XPS Document        export as PDF
File > Export > Create a Video                     render the whole deck (with timings/narration) as an MP4
File > Export > Package for CD                        bundle deck + linked media + fonts for offline/portable use
File > Save As > .pptx / .potx (template) / .pdf / .png (single slide) / .jpg / image formats (each slide as an image)
File > Options > Save > "Embed fonts in the file"          preserve custom fonts across machines
```

## Macros / VBA Basics

```vb
Sub AddSlideNumberToAll()
    Dim sld As Slide
    For Each sld In ActivePresentation.Slides
        sld.HeadersFooters.SlideNumber.Visible = True
    Next sld
End Sub
```


## Common Gotchas

- Fonts not embedded and not installed on the presenting machine silently fall back to a substitute font, breaking your carefully tuned layout — embed fonts (File > Options > Save) or export to PDF/video as a fallback for presenting on unknown machines.
- Linked (not embedded) videos/images break if the source file moves or the deck is copied to another machine without the media — use File > Export > Package for CD to bundle everything, or embed media directly.
- "With Previous" and "After Previous" animation timing can produce confusing playback order on complex slides — check the Animation Pane's sequence numbers, not just the visual layout, when troubleshooting.
- Editing individual slide layouts directly in Normal view (rather than via Slide Master) creates inconsistency across the deck — any change meant to apply everywhere belongs in Slide Master view.
- Pasting content from Word/Excel can bring inconsistent fonts/styles — use Paste Special > Use Destination Theme (or Keep Text Only) to match the deck's existing look.
- The Morph transition only works smoothly between two slides that share duplicated objects (same object, same name) — inconsistent object names/types between the two slides prevent Morph from matching them correctly.
